## Tags de Estilo/Formato (NÃO são arquétipos)

Estes tipos NÃO recebem blueprint próprio, apenas tags de estilo:

### Tags de Formato de Serviço:
- Delivery only / Dark kitchen → `_vc_style = 'dark_kitchen'`
- Drive-thru → `_vc_style = 'drive_thru'`
- Take-away / para levar → `_vc_style = 'take_away'`
- Praça de alimentação / food court → `_vc_style = 'food_court'`
- Rodízio (geral) → `_vc_service = 'rodizio'`
- Buffet livre → `_vc_service = 'buffet'`
- À la carte → `_vc_service = 'a_la_carte'`

### Tags de Experiência/Audiência:
- Restaurante familiar / kids friendly → `_vc_audience = 'family'`
- Restaurante romântico → `_vc_occasion = 'romantic'`
- Restaurante temático → `_vc_occasion = 'themed'`
- Restaurante com música ao vivo → `_vc_occasion = 'live_music'`
